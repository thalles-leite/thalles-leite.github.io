MEU APRENDIZADO

Dentre os primeiros aprendizados obtidos até então, estão o significado dos termos Unix e Bash, a importância da utilização de um SO baseado no Unix e os principais comandos Bash. Foi possível compreender também, o processo de versionamento de código utilizando o Git. E para a hospedagem dos códigos com controle de versão, aprendemos utilizar as principais ferramentas do GitHub. Em seguida, aprendemos os fundamentos da linguagem de marcação HTML e a linguagem de folha de estilos CSS.

DICAS

ALGUNS COMANDOS UNIX
COMANDO FUNÇÃO
Ctrl+Alt+t Abrir o terminal.
cd Navegação entre pastas.
ls Lista todos os arquivos e diretórios.
mkdir Cria um diretório.
rmdir Remove um diretírio.
touch Criar arquivo.
cp Copiar arquivo.
cat Abrir arquivo.
rm Apagar arquivo.
COMANDOS GIT
COMANDO FUNÇÃO
git init Transforma uma pasta em um repositório.
git add . Adiciona todos os arquivos modificados em staging.
git status Verifica o status dos arquivos do projeto.
git commit -m "Mensagem" Cria um ponto e acesso para a alteração e possui uma mensagem descrevendo as alterações feitas.
git checkout -b nome-da-branch Cria uma nova branch a partir da atual e alterna para ela.
git checkout nome-da-branch Alterna para a branch especificada.
git log Mostra os registros dos commits
git merge nome-da-branch Realiza as mesclagem das alterações da branch especificada.
